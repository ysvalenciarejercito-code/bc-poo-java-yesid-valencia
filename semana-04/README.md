# Semana 04 - Múltiples Objetos

## Descripción

Esta semana se realizo y creo la primera clase `Event` con base a la tercera actividad del dominio **"Celebraciones Especiales"** (Empresa de Eventos). Se entienden conceptos como `Constructor`, `Intancia`, `Método`, etc. 
## Objetivos Cumplidos

- [x] Identificar objetos del mundo real en el contexto del dominio. (**Actividad 1**)
- [x] Comprender y comparar la Programación Estructurada y POO. (**Actividad 2**)
- [x] Implementar la clase principal del dominio (`Event.java`). (**Actividad 3**)
- [x] Crear múltiples objetos (`Event`) con datos coherentes. (**Actividad 4**)
- [x] Documentar el análisis de objetos en el dominio.
